Bilan et Perspectives

Ce projet de deux semaines nous a permis de développer nos compétences en travail d'équipe, en gestion de projet et en développement web. Cependant l'approche collaborative adoptée n'a pas vraiment été un succès puisque plusieurs membres étaient absents pour les premières séances (très cruciales) et ne se manuifestaient pas trop. De plus même aux derniers rendez-vous fixés pour terminer le projet ils étaient en retard et il fallait les attendre plus d'une heure avant de commencer. Ils n'étaient pas non plus présents à la première séance de travail avec les encadreurs pour poser les difficultés rencontrées au groupe et aux encadreurs, ce qui nous a beaucoup porté préjudice à la fin.

Points forts : Organisation efficace, communication fluide, répartition équitable des tâches, respect des délais

Améliorations possibles : Documentation plus détaillée, tests automatisés, déploiement continu, maquettes plus complètes

Apprentissages : Travail en équipe, gestion du temps sous contrainte, développement agile, intégration de technologies web

Instructions de Déploiement

NB : VOUS DEVEZ ÊTRE CONNECTÉ À INTERNET POUR QUE L'APPLICATION FONCTIONNE!

  1. Créer un dossier projet et l’ouvrir dans votre éditeur de code.
  2. Créer et activer un environnement virtuel :
    • Sous Windows : python -m venv mon_env puis .\mon_env\Scripts\activate
    • Sous Linux : python3 -m venv mon_env puis source mon_env/bin/activate
  3. Cloner le dépôt :
    git clone https://github.com/Eudes147/PIL1_2425_28.git
  4. Aller dans le dossier du projet :
    cd PIL1_2425_28
  5. Installer les dépendances :
    pip install -r requirements.txt
  6. Configurer la base de données MySQL dans settings.py : 
    DATABASES = {
  'default': {
    'ENGINE': 'django.db.backends.mysql',
    'NAME': 'your_db_name',
    'USER': 'your_db_user',
    'PASSWORD': 'your_db_password',
    'HOST': 'your_db_host',  # 'localhost' ou '127.0.0.1' en local
    'PORT': '3306',
  }
}
  7. Appliquer les migrations :
    python manage.py makemigrations
    python manage.py migrate
  8. Créer un superutilisateur :
    python manage.py createsuperuser
  9. Démarrer le serveur de développement :
    python manage.py runserver

Manuel d'Utilisation

  1. Accueil : Au lancement du serveur, vous arrivez sur la page d'accueil.
  2. Inscription : Cliquez sur le lien Inscription dans l'en-tête. Remplissez le formulaire et validez. Vous serez redirigé vers la page de création de profil.
  3. Création du profil : Complétez les champs (nom, prénom, photo, localisation, horaires, etc.) puis cliquez sur Enregistrer le profil. Vous serez redirigé vers l'accueil secondaire.
  4. Accueil secondaire :
    • Pour accéder à la messagerie, cliquez sur Message dans l'en-tête.
    • Pour faire une demande de covoiturage, cliquez sur Faire une demande de covoiturage et remplissez le formulaire.
    • Pour proposer une offre de covoiturage, cliquez sur Faire une offre de covoiturage et remplissez le formulaire.
    • Après publication, attendez que l’algorithme de matching trouve une correspondance. Vous recevrez alors les coordonnées de votre correspondant.
  5. Liste des discussions : Retrouvez toutes vos discussions en cours et cliquez sur une discussion pour l’ouvrir.
  6. Récupération de mot de passe : Sur la page de connexion, cliquez sur <mot de passe oublié ?> pour réinitialiser votre mot de passe.
